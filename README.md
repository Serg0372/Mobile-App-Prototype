# 📱 Mobile App Prototype

Сучасний лендінг для просування мобільного застосунку, реалізований з фокусом на адаптивність, типографіку та інтерактивність. Проєкт демонструє професійний підхід до верстки багатосекційних інтерфейсів у стилі Mobile First.

## 🎯 Ключові цілі

- **Mobile First:** Реалізація архітектури для ідеального відображення на смартфонах з поступовим розширенням на планшет та десктоп.
- **Typography:** Глибоке опрацювання типографіки та декоративних елементів за допомогою CSS variables.
- **Interactive UI:** Створення форм зворотного зв'язку, модального вікна завантаження та анімованих переходів між елементами.
- **Component Structure:** Розбиття інтерфейсу на незалежні HTML-партіали для зручної підтримки та масштабування.

## 🛠 Технологічний стек

- **HTML5** — повністю семантична розмітка для кращого SEO та доступності
- **CSS3** — Flexbox, Grid, CSS variables, calc(), cubic-bezier анімації
- **Responsive** — Mobile First підхід, медіазапити (@media)
- **Vite** — оптимізована збірка активів та швидка розробка
- **SVG Sprite** — векторна графіка для іконок та логотипів

## 🚀 Технічні особливості

- **Адаптивність:** Повна підтримка пристроїв (Mobile → Tablet → Desktop)
- **Retina-оптимізація:** Використання форматів WebP та srcset (1x, 2x) для ідеальної чіткості зображень
- **Мобільне меню:** Анімоване меню з плавним виїздом та overlay
- **Modal Window:** Модальне вікно завантаження з двома планами (Free та $20) та інформацією про продукт
- **Contact Form:** Форма зворотного зв'язку з полями Name, Email, Message та CSS-валідацією
- **Team Section:** Секція команди з картками учасників та їх ролями
- **Testimonials:** Секція відгуків користувачів
- **Video Section:** Секція з демонстрацією застосунку та посиланнями на App Store і Google Play
- **Blob Backgrounds:** Декоративні SVG blob-фони з адаптивними варіантами для кожного брейкпоінту
- **SVG Graphics:** Векторна графіка для іконок та логотипів для чіткості при будь-якому масштабі
- **Scroll Buttons:** Фіксовані кнопки прокрутки вгору та вниз
